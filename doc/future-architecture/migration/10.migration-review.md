## `/core/` — 20 files

### `core/state/` — State Primitives (4 files)

| File | Exports | Used by |
|------|---------|---------|
| `state.ts` | `TOneState`, `TComponentState`, `TGlobalState` | 80+ files across entire codebase |

## Decision Options

For each section above, choose one:

- **Keep** — No move needed. Location is fine or moving has too little value.
- **Move to `/ui/`** — It's a UI component that belongs with the view layer.
- **Move to `/api/`** — It's a service or logic that belongs with the API layer.
- **Move to `/platform/`** — It's a foundational primitive (state, events, utilities).
- **Absorb** — Merge into an existing module elsewhere.
- **Delete** — No longer needed.
- **Defer** — Will be addressed in a future phase (6 or 7).
