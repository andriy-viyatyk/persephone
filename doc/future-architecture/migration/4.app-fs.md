## What

Self-contained file system implementation. `api/fs.ts` is the single source of truth for all file I/O in the application ‚Äî no delegation to external modules.

**Absorbed from `nodeUtils`:**
- Encoding-aware file read/write (BOM detection, jschardet, iconv-lite)
- `fileExistsSync()`, file deletion, directory creation, directory listing

**Absorbed from `filesModel`:**
- Init/wait pattern (fire-and-forget async init, `wait()` ensures ready)
- Path resolution: `dataPath`, `cachePath`, `cacheMiscPath` (private fields)
- Data file operations: `getDataFile`, `saveDataFile`, `deleteDataFile`, `prepareDataFile`, `dataFileName`
- Cache operations: `getCacheFile`, `saveCacheFile`, `deleteCacheFile`, `deleteCacheFiles`
- Cache-misc operations: `getCacheMiscFile`, `saveCacheMiscFile`, `deleteCacheMiscFile`, `cacheMiscFilePath`
- Binary file operations: `saveBinaryFile`, `getBinaryFile`

**Also moved (not to fs.ts):**
- `watchFile()` / `getFileStats()` ‚Üí inlined into `core/services/file-watcher.ts` (only consumer)

**Deleted modules:**
- `store/files-store.ts` ‚Äî fully absorbed
- `core/utils/node-utils.ts` ‚Äî file I/O absorbed into fs.ts, `uuid()` eliminated

## Design Decisions

1. **Full absorption** ‚Äî All logic lives in `api/fs.ts`. No delegation to external modules. Eliminates circular dependencies and init timing questions.
2. **IFileSystem unchanged** ‚Äî The public script interface stays lean. Data/cache/binary ops are on the `FileSystem` class only (not `IFileSystem`).
3. **Init timing preserved** ‚Äî Module-level `export const fs = new FileSystem()` triggers init on first import, same timing as old `filesModel.init()`.
4. **No TGlobalState** ‚Äî Plain private fields + init promise. Reactive state was unnecessary since `windowIndex` never changes after init.
5. **`fileExistsSync()`** ‚Äî Sync method on class (not on IFileSystem). Used by EventHandler, favicon-cache, file-watcher for synchronous existence checks.
6. **Naming collision** ‚Äî Files with `const fs = require("fs")` use `import { fs as appFs }` alias.
