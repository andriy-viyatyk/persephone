## Implementation Plan

### 3.1. Define IRecentFiles interface

**File:** `/src/renderer/api/types.ts` (add to existing)

```typescript
export interface IRecentFiles {
  /** List of recent file paths, most recent first. */
  readonly files: readonly string[];

  /** Add a file path to the top of the list. */
  add(filePath: string): Promise<void>;

  /** Remove a file path from the list. */
  remove(filePath: string): Promise<void>;

  /** Clear all recent files. */
  clear(): Promise<void>;
}
```

### 3.2. Implement RecentFilesApi wrapper

**File:** `/src/renderer/api/recent.ts`

```typescript
import { recentFiles } from "../store/recent-files";
import type { IRecentFiles } from "./types";

class RecentFilesApi implements IRecentFiles {
  get files(): readonly string[] {
    return recentFiles.state.get().files;
  }

  async add(filePath: string): Promise<void> {
    await recentFiles.add(filePath);
  }

  async remove(filePath: string): Promise<void> {
    await recentFiles.remove(filePath);
  }

  async clear(): Promise<void> {
    await recentFiles.clear();
  }
}

export const recent = new RecentFilesApi();
```

**Notes:**
- `files` getter reads from reactive state — always up-to-date
- `load()` is NOT exposed — it's an internal initialization step
- All mutation methods are async (they persist to disk)

### 3.3. Register on `app` root

Add `readonly recent: IRecentFiles` to `IApp` and wire in `app.ts`.

### 3.4. Write API reference

Create `../api-reference/recent.md`.
